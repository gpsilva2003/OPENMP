# Repositório de Exemplos OpenMP 

Este repositório contém os exemplos OpenMP do livro "Programação Paralela e Distribuída".
Todos os programas podem ser compilados com uso do Makefile usando o comando `make`. 
Para executar os programas compilados podem ser o comando `make run PROG=nomedoprograma`. 

Para detalhes sobre o uso dos compiladores e das variáveis de ambiente, consulte o arquivo sobre [Preparação do Ambiente](docs/ambiente.md).

Para detalhes sobre o controle de afinididade na execução das *threads*, consulte o arquivo sobre [Controle de Afinidade](docs/afinidade.md).

Para detalhes sobre o uso do escalonador de tarefas SLURM, consulte o arquivo sobre o [Escalonador SLURM](docs/slurm.md).
