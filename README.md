# Repositório de Exemplos OpenMP 

Este repositório tem os exemplos OpenMP do livro "Programação Paralela e Distribuída".
Todos os arquivos podem ser compilados com uso do Makefile usando o comando `make`. 
Para executar os programas compilados podem ser o comando `make run PROG=nomedoprograma`. 

Para detalhes sobre o uso dos compiladores e das variáveis de ambeiente, consulte o arquivo sobre [Preparação do Ambiente](docs/ambiente.md).

Para detalhes sobre o controle de afinididade na execuçaõ das *threads*, consulte o arquivo sobre [Controle de Afinidade](docs/afinidade.md).

Para detalhes sobre o uso do escalonador de tarefas SLURM, consulte o arquivo [Gerenciador SLURM](docs/slurm.md).
